# The Merry Competition

## Running

-   `npm install`
-   `npm start`

Data is saved in the `scores.json` file.

## Endpoints

-   `/` - The index page, generated by Pug
-   `/data` - The scores data in a JSON format
-   `/add/[name]/[confirm]` - Adding scores. If adding a new person, confirm must be `yes`
